# How to start

On first Run: 

    1. install Node.js (https://nodejs.org/en/download/)

    2. run "npm install -g @angular/cli"

    3. cd into /

    4. run "npm install"

    5. cd into /frontendDev

    6. run "npm install"

    7. ng b

After that:
    
    1. cd into /

    2. run "npm start"
